# Information on reservoirs

This folder contains the information on reservoirs, their historical record and quantities of interest within the model. Files in this repository are:

=> "fields_morris.csv" 
describes the quantities of interest in this analysis

=> "gages.csv"
gives basic information on the reservoirs, including the alphanumeric code

=> The 5 .csv files whose name starts with either "WY" or "ID" contain recorded reservoir inflow, outflow and storage data from Jan 1st 2009 to Dec 31st 2016. Document names corresponds to the code in "gages.csv"
